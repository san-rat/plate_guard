# PlateGuard V1 Scope

## In Scope

- Search by vehicle number
- Search by owner name
- Search by phone number
- Create promotion
- Activate promotion
- Deactivate promotion
- Add promotion usage
- Block duplicate promotion usage for the same vehicle
- Edit record
- Delete record with password
- Export CSV

## Out of Scope

- Printing
- Multi-user login
- Cloud sync
- Internet features
- Analytics dashboard
- Billing system
- SMS or email integration

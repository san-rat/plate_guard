# PlateGuard First Live Setup Checklist

Date: ____________________

Site / Shop: ____________________

Installed by: ____________________

## Before Installation

- Confirm this is the correct target computer.
- Confirm the delivery folder or installer matches the approved release.
- Confirm the user understands where backups will be stored.

## Installation

- Run the installer, or copy the portable delivery folder to the target machine.
- Confirm PlateGuard opens normally.
- Confirm a desktop shortcut exists or create one with `Create-Desktop-Shortcut.ps1`.

## Initial App Setup

- Open `Settings`.
- Enter the real shop name.
- Change the default delete password from `admin` to the live password.
- If needed, enter a full export folder path.
- Save settings successfully.

## Functional Go-Live Check

- Create the first active promotion.
- Search by vehicle number.
- Save one sample or real usage record.
- Open `History` and confirm the record is visible.
- Export a CSV and confirm the file is created.
- Confirm staff know which password is used for deletion.

## Backup Readiness

- Show the admin where `%LOCALAPPDATA%\PlateGuard\plateguard.db` is stored.
- Confirm who will make backups.
- Confirm where backup copies will be kept.

## Sign-Off

- App opens correctly
- Settings saved correctly
- Promotion created
- Usage record saved
- CSV export verified
- Backup location explained
- Delete password changed from default
